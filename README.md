# 🌍 Nostalgic AI – Historical Journey Generator

Nostalgic AI is an interactive Streamlit application that allows users to travel through history using artificial intelligence. By entering a historical year, users can explore major events, choose a unique perspective (such as a journalist, witness, or participant), and experience immersive AI-generated stories, images, and audio narrations.

This project combines multiple AI services to transform historical facts into engaging, first-person storytelling experiences.

---

## 🚀 Features

- 🔍 Search major historical events for any year
- 👥 Explore events from multiple perspectives
- 📖 Generate immersive first-person historical narratives
- 🖼️ Create AI-generated historical images
- 🔊 Convert stories into audio narration
- 🎨 Customize image style, mood, and aspect ratio
- 📊 Track completed historical journeys
- 🏆 Achievement badges for user engagement
- ⚡ Quick-select popular historical years
